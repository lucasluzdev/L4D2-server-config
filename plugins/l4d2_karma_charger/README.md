Props to Zed Time, because it showed how Slow Motion could be used in L4D2.


This Plugin checks Charger Victims for being carried over really high edges...


If such a case is found, there is a Slowmotion effect on the server, a Sound is played, and a chat notification depicts Chargers glory.



Convars:

l4d2_karma_charge_version
l4d2_karma_charge_height (default 475) - what height triggers Karma Charges
l4d2_karma_charge_slowtime (default 1.5) - how long does Slowmotion last
l4d2_karma_charge_enabled - toggle on/off
l4d2_karma_charge_notify (default 1) - toggle chat announcement in server mode
