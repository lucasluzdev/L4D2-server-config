Cvars:
tankstuck_fix_enable (default "1") - Enables or disables the Tank Stuck Fix plugin.
tankstuck_check_interval (default "5") - Sets the interval (in seconds) to check if the Tank is stuck.
tankstuck_slap_distance_initial (default "150.0") - Initial distance used to push Survivors away from the Tank if stuck.
tankstuck_slap_max_distance (default "300.0") - Maximum distance Survivors are pushed away when the Tank is stuck.
tankstuck_slap_power (default "200.0") - Force applied to Survivors when they are pushed away from a stuck Tank.
tankstuck_slap_distance_increase (default "10.0") - Amount by which the slap distance increases with each check interval.
tankstuck_slap_vertical_multiplier (default "1.2") - Multiplier for the upward force applied to Survivors when pushed by the Tank.
