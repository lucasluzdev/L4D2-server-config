So, as said above, it disables Versus scoring when Tanks and/or Witches spawn so whenever players choose to rush instead of fighting, their scores will no longer tracked beyond the distance the BI spawned. I only added support for multiple tanks and witches.

NOTE: Requires Left 4 Downtown

BUGS:
